Q-1. Write an SQL query to show only odd rows from a table.

Query:

Select *
from Worker
where mod(worker_id,2) <> 0;

Q-2. Write an SQL query to fetch intersecting records of two tables.

Query:

(SELECT * FROM Worker)
INTERSECT
(SELECT * FROM WorkerClone);

Q-3. Write an SQL query to show the current date and time.

Query:

Select curdate();//return date

Select now();//returns the current date and time

Q-4. Write an SQL query to show the top n (say 10) records of a table.

Query:

Select *
from worker
order by salary desc
LIMIT 10;

Q-5. Write an SQL query to determine the nth (say n=5) highest salary from a table.

Query:

Select *
from worker
order by salary desc
LIMIT 1 OFFSET 4;

Q-6. Write an SQL query to fetch the list of employees with the same salary.

Query:

Select *
from worker w1,worker w2
where w1.worker_id <> w2.worker_id and w1.salary=w2.salary;

Q-7. Write an SQL query to show the second highest salary from a table.

Query:

Select max(salary)
from worker
where salary NOT IN (Select max(salary) from worker);

Q-8. Write an SQL query to fetch the departments that have less than five people in it.

Query:

Select count(worker_id)
from Worker
group by department
having count(worker_id) < 5;

Q-9. Write an SQL query to show all departments along with the number of people in there.

Query:

Select department, count(Department)
from Worker
group by department;

Q-10. Write an SQL query to print the name of employees having the highest salary in each department.

Query:

SELECT w.first_name, w.department, w.salary
FROM Worker w
WHERE w.salary = (
  SELECT MAX(salary)
  FROM Worker
  WHERE department = w.department
);
